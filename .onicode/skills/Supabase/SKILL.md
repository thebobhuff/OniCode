# Supabase

You have access to the Supabase CLI and API. Use these commands to interact with Supabase projects.

## Commands

- `supabase init` — Initialize a new Supabase project
- `supabase start` — Start local Supabase services
- `supabase db push` — Push migrations to the linked project
- `supabase link --project-ref <ref>` — Link to a remote project
- `supabase gen types typescript --project-id <ref>` — Generate TypeScript types
